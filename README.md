# 龙华基层治理平台（digital-twin-longhua）

本系统包含，消防通道，垃圾桶监测，事件中心，老人一键呼叫等功能。

## 运行

```bash
git clone  https://github.com/forever25/digital-twin-longhua.git
cd digital-twin-longhua
pnpm i
pnpm dev
```
